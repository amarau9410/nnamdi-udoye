# Nnamdi Udoye Portfolio

Professional single-page portfolio website for Nnamdi Udoye, Salesforce Administrator.

## Included Files

- `index.html`: Main website
- `favicon.svg`: Site icon
- `Nnamdi_Udoye_Salesforce_CV10-03-2026.docx`: Downloadable CV
- `.nojekyll`: Ensures static files are served correctly on GitHub Pages

## Local Preview

1. Open `index.html` in your browser.

## Deploy To GitHub Pages

1. Push this repository to GitHub.
2. Open repository `Settings` -> `Pages`.
3. Under `Build and deployment`, set `Source` to `Deploy from a branch`.
4. Select branch `main` and folder `/ (root)`.
5. Click `Save`.
6. Wait for deployment, then open:
	`https://amarau9410.github.io/nnamdi-udoye/`
