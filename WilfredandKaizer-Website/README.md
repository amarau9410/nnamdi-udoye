# Wilfred and Kaizer Holding Co Website

[![Deploy static site to GitHub Pages](https://github.com/amarau9410/nnamdi-udoye/actions/workflows/deploy-pages.yml/badge.svg)](https://github.com/amarau9410/nnamdi-udoye/actions/workflows/deploy-pages.yml)

Premium single-page company website for Wilfred and Kaizer Holding Co Ltd (IT and Salesforce Consultancy and ISV).

## Deployment Status

- Workflow: [Deploy static site to GitHub Pages](https://github.com/amarau9410/nnamdi-udoye/actions/workflows/deploy-pages.yml)
- Source directory published: `WilfredandKaizer-Website`
- Custom domain: `wilfredandkaizer.co.uk`
- Live URL: [https://wilfredandkaizer.co.uk](https://wilfredandkaizer.co.uk)

## Files

- `index.html`: Main website page
- `assets/wk-logo.png`: Company logo
- `CNAME`: Custom domain for GitHub Pages (`wilfredandkaizer.co.uk`)
- `.nojekyll`: Ensures static assets are served as expected on GitHub Pages

## Design Direction

The layout is inspired by the Asentus visual language and uses sophisticated typography, structured sections, premium cards, and cinematic imagery.

## Local Preview

Open `index.html` in a browser.

## Deploy to GitHub Pages (WilfredandKaizer-Website Repo)

1. Push these files to the root of your `WilfredandKaizer-Website` repository.
2. In repository settings, open `Pages`.
3. Set source to `GitHub Actions`.
4. Ensure the workflow `Deploy static site to GitHub Pages` completes successfully.
5. Save and wait for deployment.
6. Confirm DNS points to GitHub Pages and verify `wilfredandkaizer.co.uk` resolves.
